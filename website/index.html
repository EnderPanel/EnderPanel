Install EnderPanel

Run this command in your terminal. Requires Docker to be installed.

bash 
# Install EnderPanel
curl -sSL https://enderpanel.space/install.sh | bash

Supports Ubuntu, Debian, Fedora and macOS.

powershell 
# Install EnderPanel
irm https://enderpanel.space/install.ps1 | iex
